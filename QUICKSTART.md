# 🚀 Quick Start Guide

## Installation (2 minutes)

1. **Download all plugin files** to a folder on your computer
2. **Open Figma Desktop** (must use desktop app, not browser)
3. Go to: **Menu → Plugins → Development → Import plugin from manifest...**
4. Select the `manifest.json` file
5. Done! ✅

## First Use (1 minute)

1. **Open the plugin**: Menu → Plugins → Development → Smart Find & Replace
2. **Type what to find**: e.g., "Hello kitty"
3. **Type replacement**: e.g., "Hello mama"
4. **Keep these ON**:
   - ✅ Match whole words only
   - ✅ Match exact phrase
5. **Click "Find Matches"** to preview
6. **Click "Replace All"** to apply

## Common Scenarios

### Replace phrase in specific frames only
1. **Select** the frames you want to change
2. **Enable** "Search in selection only"
3. Find and replace as normal

### Replace a word everywhere
1. **Turn OFF** "Match exact phrase"
2. **Keep ON** "Match whole words only"
3. This replaces the word in all contexts

### Case-sensitive replacement
1. **Enable** "Case sensitive"
2. Now "API" won't match "api"

## Tips
- Always **preview first** (Find Matches button)
- Use **selection mode** for targeted changes
- Check the **match counter** before replacing
- Plugin **auto-selects** matching layers

## Troubleshooting
- **Not finding text?** → Turn off "Case sensitive"
- **Too many matches?** → Enable "Match exact phrase"
- **Plugin not showing?** → Must use Figma Desktop app

---
**You're ready to go! Happy designing! 🎨**
